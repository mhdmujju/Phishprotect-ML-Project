# Phishprotect-ML-Project

## 📌 Project Overview
Phishprotect-ML-Project is a Machine Learning based phishing website detection system.  
The project analyzes URLs and predicts whether a website is legitimate or phishing.

---

## 🚀 Features
- Phishing URL Detection
- Machine Learning Model
- Dataset Processing
- Web Interface using Flask
- CSV Dataset Support

---

## 🛠️ Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

---


---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/mhdmujju/Phishprotect-ML-Project.git
```

2. Install required libraries

```bash
pip install -r requirements.txt
```

3. Run the Flask app

```bash
python app.py
```
